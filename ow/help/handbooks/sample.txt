Use mail aliases to send messages
to groups of people whom you e-mail
regularly.  For example, an alias called
"workgroup" could include all the people
in your group.  This way, you do not need
to type in each person's name every time
you send e-mail to your group.
 
You can create mail aliases either by
modifying the file /etc/aliases, or by
adding them to the file .mailrc in your
home directory.  (You can modify .mailrc
from Mail Tool.)  Aliases in /etc/aliases
can be used by anyone; aliases in .mailrc
can be used only by you.  But you must
have superuser privileges to change
/etc/aliases.

Utilisez les alias de courrier 
électronique pour envoyer des messages
aux groupes de personnes avec lesquels
vous communiquez régulièrement. Par
exemple, l'alias "groupe" peut englober 
toutes les personnes de votre groupe de
travail. Cela vous évite d'indiquer le 
nom de chaque personne lorsque vous 
devez envoyer un message à votre groupe. 

Vous pouvez créer des alias de courrier
électronique en modifiant le fichier
/etc/aliases ou en ajoutant ces alias
au fichier .mailrc dans votre répertoire
utilisateur. (Vous pouvez modifier
.mailrc à partir de la Messagerie.) Les
alias se trouvant dans le fichier 
/etc/aliases peuvent être utilisés par
tous ; les alias du fichier .mailrc
ne peuvent être utilisés que par vous.
Vous devez cependant avoir les privilèges 
de superutilisateur pour modifier le
fichier /etc/aliases.
